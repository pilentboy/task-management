# Task Management App

A modern and lightweight task management application built with TypeScript, Vite, and Tailwind CSS. This project focuses on modular design, reusable components, and performance optimization.

## Features
- **Modular Components**: Clean and reusable code for tasks, profiles, and more.
- **Modern Styling**: Styled using Tailwind CSS for a responsive and sleek design.
- **TypeScript**: Ensures type safety and reduces runtime errors.
- **Fast Build**: Powered by Vite for blazing-fast development.
